# EasySalaire-gestion-Paie-C
## Description du projet
EasySalaire est une application console développée en Langage C permettant d'automatiser le calcul des salaires mensuels (Brut et Net) et la gestion des fiches de paie pour une entreprise (jusqu'à 50 employés).

## Fonctionnalités
-Ajout d'amployés** avec saisie dans des heures normales est supplèmentaires.
-Calcul automatique du salaire Brut** avec majoration de 150% sur les heures supplèmentaires.
-Calcul automatique du salaire Net** après déduction des taxes (15%).
-Attribution d'une prime d'assiduité** de 200 DT si l'employé dépasse 160 heures de travail.
-Affichage complet du bulletin de paie** au format console.

## Concepts Algorithmiques Utilisés
-Tableaux parallèles 1D (liaison des données par indice).
-Passage de paramètres par valeur et par pointeur ('int *n').
-Fonctions modulaires avec types de retour ('float','void').
-Structures de controle conditionnelles ('if', 'switch') et boucles ('do...while' , 'for').
